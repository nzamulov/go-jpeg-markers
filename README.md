# go-jpeg-markers

Simply native tool for fetch all markers from JPEG image.
Only Golang required, no any external deps.

## How to use as a CLI?

### Just do it
```go
make && ./gjm-cli -p <PATH_TO_FILE or LINK_FROM_INTERNET>
```
